# Tiny Creator Lab 🏗️

## About This Project

This project is focused on AI-generated miniature world videos where tiny workers build creative scenes like luxury objects, houses, and fantasy environments.

---

## 🧠 Concept

Tiny AI workers performing construction tasks in giant real-world environments.

Example ideas:
- Tiny workers building watermelon house 🍉
- Tiny workers building castle in teacup ☕
- Tiny workers building luxury eyewear 👓

---

## 🛠️ Tools Used

- ChatGPT (Prompt creation & scripting)
- Gemini (Ideas, thumbnails, creative generation)
- Google Flow (Image & video generation)
- CapCut (Video editing)
- Canva (Posters & thumbnails)

---

## ⚙️ Workflow

1. Idea generated using ChatGPT / Gemini  
2. Prompts created in ChatGPT  
3. Images generated using Google Flow  
4. Video generated using Google Flow  
5. Thumbnail created using Gemini  
6. Final editing in CapCut  
7. Posted on Instagram (Tiny Creator Lab)
